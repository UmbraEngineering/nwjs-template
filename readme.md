
# nwjs-template

## Init

```bash
$ git clone git@github.com:UmbraEngineering/nwjs-template your-project
$ cd your-project

# Clear out the template's history ...
$ rm -rf .git

# ... so you can start your own
$ git init

# Install dependencies
$ npm install
```

## Running

```bash
$ nw .
```
